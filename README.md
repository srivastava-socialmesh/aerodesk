# AeroDesk - Private Aviation Marketplace Platform

A comprehensive SaaS platform for private aviation charter management, empty leg marketplace, and corporate travel coordination.

## 🚀 Features

- **Multi-tenant Architecture**: 6 user roles (Admin, Operator, Corporate, Travel Agency, Hotel, Customer)
- **Charter RFQ Marketplace**: Request and bid on private charters
- **Empty Leg Deals**: Discounted repositioning flights
- **Seat Marketplace**: Book individual seats on flights
- **Corporate Travel Desk**: Multi-level approval workflows
- **Hotel Integration**: Accommodation booking for flight crews and passengers
- **Compliance Engine**: GST verification and audit trails
- **Real-time Analytics**: Platform intelligence and reporting

## 🛠️ Tech Stack

- **Framework**: Next.js 14 with App Router
- **Language**: TypeScript
- **Database**: Supabase (PostgreSQL)
- **Authentication**: Supabase Auth
- **Styling**: Tailwind CSS + shadcn/ui
- **Forms**: React Hook Form + Zod
- **Charts**: Recharts
- **Deployment**: Vercel
